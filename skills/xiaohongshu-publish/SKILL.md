# 小红书发布技能

将内容发布到小红书平台，支持图文笔记发布。

## 工作流程

1. **内容适配** - 转换为小红书风格
2. **生成封面** - 选择或生成封面图
3. **优化标题** - 符合小红书调性
4. **生成标签** - 热门标签推荐
5. **发布笔记** - 提交到平台

## 小红书特点

| 特点 | 说明 |
|------|------|
| 图文结合 | 3-9 张图片效果最佳 |
| 标题党 | 吸引眼球的标题 |
| 标签 | #标签 增加曝光 |
| 表情 | emoji 增加亲和力 |

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| `content` | 是 | 文章内容 |
| `images` | 是 | 图片列表（3-9张） |
| `title` | 否 | 标题，可自动生成 |
| `tags` | 否 | 标签，可自动推荐 |
