# 小红书内容规范

## 笔记要求

| 项目 | 限制 |
|------|------|
| 标题 | 5-20 字 |
| 正文 | 最多 1000 字 |
| 图片 | 1-9 张 |
| 标签 | 最多 10 个 |

---

## 图片规范

| 项目 | 建议 |
|------|------|
| 比例 | 3:4（竖图）最佳 |
| 尺寸 | 1242×1660 px |
| 格式 | JPG/PNG |
| 大小 | ≤ 20 MB |

---

## 内容禁忌

- 禁止虚假宣传
- 禁止敏感内容
- 禁止诱导互动
- 禁止侵权内容
