# 爆款笔记公式

## 标题公式

### 公式 1：数字 + 效果
```
❌ 如何减肥
✅ 30天瘦10斤，我只做了这3件事
```

### 公式 2：痛点 + 解决
```
❌ 面试技巧
✅ 面试总是挂？这5个问题回答好
```

### 公式 3：身份 + 经历
```
❌ 留学经验
✅ 985硕士留学英国，真实体验分享
```

---

## 内容结构

```markdown
# 吸睛标题

## 开头（痛点/悬念）
制造好奇，引发共鸣

## 核心内容
3-5 个干货点
每个点配一张图

## 结尾（互动引导）
❤️ 觉得有用点个赞
💬 评论区说说你的看法
```

## 互动技巧

1. **结尾提问**：引导评论
2. **@好友**：增加传播
3. **收藏福利**：引导收藏
4. **评论区互动**：提高权重
