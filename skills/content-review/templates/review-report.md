# 审核报告模板

## 文本格式

```
═══════════════════════════════════════
内容审核报告
═══════════════════════════════════════

综合评分：85/100 ✅

【严重问题】0 个
【警告】2 个
【建议】3 个

───────────────────────────────────────
警告
───────────────────────────────────────
1. [敏感词] 第 3 段发现疑似敏感词"xxx"，建议替换
2. [格式] 标题长度超过 30 字，建议精简

───────────────────────────────────────
建议
───────────────────────────────────────
1. [拼写] "的/得/地"使用建议检查
2. [格式] 段落之间建议增加空行
3. [链接] 发现 2 个外部链接，建议验证

═══════════════════════════════════════
```

---

```json
{
  "score": 85,

      }
    ],
    "suggestion": []
  },
  "summary": {
    "totalWords": 1250,
    "totalParagraphs": 8,
    "totalLinks": 2
  }
}
```
