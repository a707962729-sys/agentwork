# 格式规范

## 标题规范

| 规则 | 说明 |
|------|------|
| 长度 | 5-30 字为宜 |
| 格式 | 不使用全角标点 |
| 标点 | 结尾不加标点 |

## 段落规范

| 规则 | 说明 |
|------|------|
| 长度 | 每段 100-300 字 |
| 间距 | 段落间空一行 |
| 缩进 | 不使用首行缩进 |

## 标点规范

| 问题 | 正确 | 错误 |
|------|------|------|
| 连续标点 | …… | 。。。 |
| 全角半角 | ，| , |
| 引号 | "内容" | "内容" |

## 常见格式错误

1. **全角半角混用**
   - 错误：Hello，World
   - 正确：Hello，World（中文语境）

2. **多余空格**
   - 错误：这是 一段 文字
   - 正确：这是一段文字

3. **标点粘连**
   - 错误：你好。我来了。
   - 正确：你好。我来了。
