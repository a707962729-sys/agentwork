# 敏感词词库

## 敏感词分类

### 政治类
- 涉及国家领导人的不当表述
- 政治敏感事件关键词
- 违禁政治术语

### 色情低俗类
- 性暗示词汇
- 低俗用语
- 成人内容关键词

### 暴力恐怖类
- 暴力行为描述
- 恐怖主义相关
- 危险行为诱导

### 广告营销类
- 虚假宣传词汇
- 诱导点击用语
- 违禁广告词

### 其他类
- 宗教敏感词
- 民族歧视词
- 侵权词汇

---

## 处理方式

| 级别 | 处理 |
|------|------|
| **严重** | 禁止发布，必须修改 |
| **警告** | 提示风险，建议修改 |
| **建议** | 可选优化 |

## 自定义词库

可在配置文件中添加自定义敏感词：

```json
{
  "customSensitiveWords": [
    "竞品名称",
    "内部代号"
  ]
}
```
