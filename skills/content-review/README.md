# Content Review 技能

内容审核技能，用于检测敏感词、检查格式规范、生成审核报告。

## 快速开始

```bash
# 安装（如需要）
npm install -g content-review-cli

# 基本使用
content-review --text "待审核内容"

# 审核文件
content-review --file article.md
```

## 功能特性

- 🔍 **敏感词检测** - 多类别敏感词库
- 📝 **格式检查** - 标题、段落、标点规范
- ✏️ **错别字检测** - 常见错误识别
- 🔗 **链接验证** - 外部链接有效性检查
- 📊 **审核报告** - 分级问题列表 + 修改建议

## 使用场景

1. **自媒体内容发布前审核**
2. **公众号文章合规检查**
3. **博客文章质量把关**
4. **营销文案敏感词过滤**

## 输出示例

```
综合评分：85/100 ✅
【严重问题】0 个 | 【警告】2 个 | 【建议】3 个

警告:
1. [敏感词] 第 3 段发现疑似敏感词"xxx"
2. [格式] 标题长度超过 30 字

建议:
1. [拼写] "的/得/地"使用建议检查
2. [格式] 段落之间建议增加空行
```

## 配置

在 `~/.content-review/config.json` 中配置自定义词库和规则。

详见 SKILL.md 获取完整文档。
