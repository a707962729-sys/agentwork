# 内容审核技能

全面的内容审核能力，包括敏感词检测、格式规范、错别字检查。

## 工作流程

1. **内容解析** - 读取文本或文件
2. **敏感词检测** - 政治色情暴力等
3. **格式检查** - 标题、段落、标点
4. **错别字检测** - 常见错误识别
5. **生成报告** - 问题分级、修改建议

## 检查类别

| 类别 | 说明 |
|------|------|
| `sensitive` | 敏感词检测 |
| `format` | 格式规范 |
| `spelling` | 错别字 |
| `links` | 链接有效性 |

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| `text` | 二选一 | 待审核文本 |
| `file` | 二选一 | 文件路径 |
| `strict` | 否 | 严格模式，默认 false |
| `skip` | 否 | 跳过的检查类别 |
