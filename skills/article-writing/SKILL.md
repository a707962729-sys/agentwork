# 文章写作技能

根据主题和要求，撰写高质量的文章内容。

## 工作流程

1. **理解需求** - 分析主题、目标读者、风格、字数要求
2. **生成大纲** - 先输出结构供用户确认
3. **撰写正文** - 按大纲逐节展开
4. **优化润色** - 检查逻辑、语言、格式

## 支持的文章类型

| 类型 | 说明 | 适合场景 |
|------|------|----------|
| 科普文章 | 通俗易懂解释概念 | 知识传播、公众号 |
| 技术博客 | 专业深度技术分享 | 技术社区、个人博客 |
| 商业分析 | 数据驱动商业洞察 | 报告、白皮书 |
| 新闻资讯 | 客观及时事件报道 | 媒体平台 |
| 观点评论 | 鲜明立场深度分析 | 专栏、社论 |

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| `topic` | 是 | 文章主题 |
| `style` | 否 | 写作风格：专业/轻松/幽默，默认专业 |
| `length` | 否 | 目标字数，默认 800 |
| `outline` | 否 | 自定义大纲，不填则自动生成 |
| `audience` | 否 | 目标读者，影响用词和深度 |
