# 文章输出模板

## 标准输出格式

```json
{
  "title": "文章标题",
  "summary": "100字以内的摘要",
  "content": "# 标题\n\n正文内容...\n\n## 二级标题\n\n更多内容...",
  "wordCount": 1234,
  "sections": ["章节1", "章节2", "章节3"],
  "tags": ["标签1", "标签2"],
  "readingTime": "预计阅读5分钟"
}
```

## Markdown 正文格式规范

- 标题：使用 `#` `##` `###`
- 强调：使用 **加粗** 和 *斜体*
- 列表：使用 `-` 或 `1.`
- 代码：使用 ` ```语言 ` 代码块
- 引用：使用 `>` 引用块
- 链接：`[文字](URL)`

## 字数估算

| 文章类型 | 推荐字数 | 阅读时间 |
|----------|----------|----------|
| 公众号 | 1500-3000 | 5-10分钟 |
| 技术博客 | 2000-5000 | 8-15分钟 |
| 深度分析 | 3000-8000 | 10-25分钟 |
| 短文/动态 | 300-800 | 1-3分钟 |
