# 大纲输出模板

## JSON 格式

```json
{
  "title": "文章标题建议",
  "summary": "文章摘要（100字以内）",
  "sections": [
    {
      "id": "section-1",
      "title": "章节标题",
      "points": ["要点1", "要点2", "要点3"],
      "estimatedWords": 300
    }
  ],
  "totalEstimatedWords": 1500,
  "keywords": ["关键词1", "关键词2"],
  "style": "专业",
  "platform": "微信公众号"
}
```

---

## Markdown 格式

```markdown
# 文章大纲：[主题]

## 基本信息
- **标题**：建议标题
- **摘要**：文章摘要
- **预估字数**：1500 字
- **目标平台**：微信公众号

## 结构大纲

### 1. 开篇引入（200字）
- 背景铺垫
- 引出主题
- 吸引读者

### 2. 核心内容（800字）
#### 2.1 章节一（300字）
- 要点 A
- 要点 B

#### 2.2 章节二（300字）
- 要点 A
- 要点 B

#### 2.3 章节三（200字）
- 要点 A

### 3. 总结收尾（300字）
- 核心回顾
- 延伸思考

## 关键词
#标签1 #标签2 #标签3
```
