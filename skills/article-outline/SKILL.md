# 文章大纲设计技能

根据主题和上下文，生成结构化的文章大纲。

## 工作流程

1. **理解主题** - 分析用户需求
2. **确定文体** - 选择合适的结构
3. **规划章节** - 设计逻辑顺序
4. **细化要点** - 每章节核心内容
5. **输出大纲** - 结构化呈现

## 支持的文体

| 文体 | 结构特点 |
|------|----------|
| 科普文章 | 概念→原理→应用→总结 |
| 技术博客 | 背景→原理→实现→总结 |
| 观点评论 | 现象→观点→论据→结论 |
| 商业分析 | 背景→数据→洞察→建议 |

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| `topic` | 是 | 文章主题 |
| `style` | 否 | 写作风格 |
| `platform` | 否 | 目标平台 |
| `length` | 否 | 目标字数 |
