# 搜索结果输出模板

## 标准输出格式

```json
{
  "query": "搜索关键词",
  "engine": "使用的搜索引擎",
  "total": 10,
  "freshness": "week",
  "results": [
    {
      "rank": 1,
      "title": "结果标题",
      "url": "https://...",
      "snippet": "摘要内容...",
      "source": "来源网站",
      "publishedDate": "2024-01-15",
      "relevanceScore": 0.95
    }
  ],
  "suggestions": ["相关搜索词1", "相关搜索词2"]
}
```

---

## Markdown 输出格式

```markdown
# 搜索结果：[关键词]

共找到 10 条结果（最近一周）

## 🔥 高相关

### 1. [标题](URL)
> 摘要内容...
>
> 来源：xxx.com | 发布：2024-01-15

### 2. [标题](URL)
> 摘要内容...

## 📚 相关资源

- [相关链接1](URL)
- [相关链接2](URL)
```

---

## 聚合输出

当需要深度调研时，输出聚合结果：

```markdown
# 调研报告：[主题]

## 关键发现
1. 发现1
2. 发现2
3. 发现3

## 来源汇总
| 来源 | 数量 | 质量评估 |
|------|------|----------|
| 官方文档 | 3 | ⭐⭐⭐⭐⭐ |
| 技术博客 | 5 | ⭐⭐⭐⭐ |
| 新闻媒体 | 2 | ⭐⭐⭐ |

## 核心观点
- 观点1（来源A, B）
- 观点2（来源C, D）
- 观点3（来源E）
```
