# Markdown 格式规范

## 支持的语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式
```markdown
**加粗** *斜体* ~~删除线~~
```

### 列表
```markdown
- 无序列表
- 无序列表

1. 有序列表
2. 有序列表
```

### 引用
```markdown
> 引用内容
```

### 链接和图片
```markdown
[链接文字](URL)
![图片说明](图片URL)
```

### 代码块
````markdown
```javascript
console.log("Hello");
```
````

---

## 微信公众号特殊格式

### 摘要
```markdown
<!-- 摘要 -->
这是文章摘要，会显示在推送列表中
```

### 封面图
```markdown
<!-- 封面 -->
![cover](cover.jpg)
```

### 作者
```markdown
<!-- 作者 -->
作者：张三
```

---

## 样式建议

| 元素 | 建议 |
|------|------|
| 标题 | 使用二级标题为主 |
| 段落 | 每段 100-200 字 |
| 图片 | 宽度 900px，高度不超过 2000px |
| 引用 | 用于金句或重要提示 |
