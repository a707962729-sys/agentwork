# 任务拆解技能

## 功能
将用户输入的复杂任务拆解为多个可执行的子任务步骤。

## 使用场景
- 用户表达一个复杂的工作需求
- 需要多步骤协作完成的任务
- 需要确定执行顺序的任务

## 拆解原则
1. **MECE原则** - 相互独立，完全穷尽
2. **最小可执行单元** - 每个子任务有明确的输入输出
3. **依赖关系明确** - 清晰标注步骤间的依赖
4. **角色匹配** - 每个子任务分配给合适技能的Agent

## 输出格式
```json
{
  "steps": [
    {
      "id": "step-1",
      "title": "步骤标题",
      "skill": "使用的技能",
      "dependsOn": [],
      "input": {}
    }
  ]
}
```
