# Image Gen 技能

AI 图片生成技能，支持封面图、配图生成，多种艺术风格可选。

## 快速开始

```bash
# 生成封面图
image-gen --type cover --prompt "科技主题，蓝色调，未来感"

# 生成插图
image-gen --prompt "团队协作场景" --style illustration

# 批量生成
image-gen --prompt "四季风景" --count 4 --variation
```

## 支持类型

- 📰 **封面图** - 文章、视频、社交媒体封面
- 🖼️ **插图** - 文章配图、教程步骤图
- 🎨 **图标** - 装饰元素、UI 图标
- 🌅 **背景** - 网页背景、演示背景

## 艺术风格

| 风格 | 描述 |
|------|------|
| realistic | 写实摄影 |
| illustration | 手绘插画 |
| flat | 扁平设计 |
| watercolor | 水彩画 |
| cyberpunk | 赛博朋克 |
| chinese | 中国风 |
| minimalist | 极简主义 |
| 3d | 3D 渲染 |

## 使用场景

1. **自媒体配图** - 公众号、博客文章插图
2. **视频封面** - YouTube、B 站封面
3. **社交媒体** - 微博、小红书配图
4. **产品展示** - 产品概念图、宣传图

## 输出示例

```

文件：cover_20240115_001.png

风格：cyberpunk

```

## 提示词技巧

```
[主体] + [场景] + [风格] + [色调] + [细节]

例："一只猫坐在窗台上，水彩风格，暖色调，阳光透过窗户"
```
