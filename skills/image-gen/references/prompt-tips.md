# 提示词技巧

## 好的提示词结构

```
[主体] + [场景/背景] + [风格] + [色调] + [细节]

示例：
"一只猫 (主体) 坐在窗台上 (场景)，水彩风格 (风格)，暖色调 (色调)，阳光透过窗户 (细节)"
```

---

## 风格修饰词

### 高质量
- 4k, highly detailed, professional
- sharp focus, detailed texture
- studio lighting, professional photography

### 氛围
- warm lighting, cinematic, dramatic
- soft light, dreamy, ethereal
- dark, moody, mysterious

### 构图
- centered, rule of thirds, wide angle
- close-up, macro, portrait
- aerial view, bird's eye view

## 常见问题解决

| 问题 | 解决方案 |
|------|----------|
| 人物畸形 | 添加 "symmetrical face, anatomically correct" |
| 文字乱码 | 避免在提示词中要求文字 |
| 色彩暗淡 | 添加 "vibrant colors, colorful" |
| 构图混乱 | 添加 "centered, clean composition" |
